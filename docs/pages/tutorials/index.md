# Tutorials

Tutorials walk you through complete workflows from start to finish. Each one is designed to teach a specific skill by doing. Follow the steps in order and you will have a working result at the end.

**New to [Kedro](https://docs.kedro.org/) or [Azure ML](https://learn.microsoft.com/en-us/azure/machine-learning/)?** The Getting Started tutorial assumes basic familiarity with both. Links to prerequisite reading are included in each tutorial.

- [**Getting Started**](getting-started.md): Install the plugin, configure a workspace, and submit your first Kedro pipeline to Azure ML managed compute.
- [**Migrate an Existing Project**](migrate-existing-project.md): Add Azure ML support to a Kedro project you already have.
